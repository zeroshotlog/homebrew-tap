# zeroshotlog Homebrew Tap

Homebrew tap for zeroshotlog apps.

## Installation

```bash
brew tap zeroshotlog/tap
```

## Available Casks

### Portman

Local port management tool for macOS.

```bash
brew install --cask portman
```

### Orchestra

AI coding agent session monitor for macOS.

```bash
brew install --cask orchestra
```

## Updating

```bash
brew update
brew upgrade --cask portman
brew upgrade --cask orchestra
```
